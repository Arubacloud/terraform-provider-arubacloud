# arubacloud_key

Manages an ArubaCloud encryption key resource within a Key Management Service (KMS). Keys are used for encryption and decryption operations.

## Example Usage

```terraform
resource "arubacloud_kms" "example" {
  name       = "production-kms"
  project_id = arubacloud_project.example.id
}

resource "arubacloud_key" "example" {
  name       = "aes-encryption-key"
  project_id = arubacloud_project.example.id
  kms_id     = arubacloud_kms.example.id
  algorithm  = "Aes"
}

resource "arubacloud_key" "rsa_key" {
  name       = "rsa-encryption-key"
  project_id = arubacloud_project.example.id
  kms_id     = arubacloud_kms.example.id
  algorithm  = "Rsa"
}
```

### Required

- `name` (String) Key name
- `project_id` (String) ID of the project this Key belongs to

- `algorithm` (String) Encryption algorithm (e.g., "Aes", "Rsa")

### Optional

- `description` (String) Key description - optional, not returned by API

- `uri` (String) Key URI
- `status` (String) Key status
