# arubacloud_kmip

Manages an ArubaCloud KMIP endpoint resource. KMIP provides a standardized protocol for key management operations.

**Note:** KMIP creation may return validation errors depending on KMS configuration or feature availability.

## Example Usage

```terraform
resource "arubacloud_kms" "example" {
  name       = "production-kms"
  project_id = arubacloud_project.example.id
}

resource "arubacloud_kmip" "example" {
  name       = "production-kmip"
  project_id = arubacloud_project.example.id
  kms_id     = arubacloud_kms.example.id
}
```
